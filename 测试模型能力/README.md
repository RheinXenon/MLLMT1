# Lingshu-7B 模型测试工具集

本文件夹包含了测试 Lingshu-7B 模型的所有脚本。

## 使用说明

### 前提条件
确保你在项目根目录 (`lingshu-7b`) 下已经准备好：
- `models/Lingshu-7B/` - 模型文件目录
- `test_images/` - 测试图像目录

### 脚本说明

1. **1_下载模型.py** - 从 HuggingFace 下载 Lingshu-7B 模型
2. **2_测试模型.py** - 测试纯文本对话功能
3. **3_测试多模态模型.py** - 测试图像分析功能

### 运行方式

进入 `测试模型能力` 文件夹后，直接运行脚本：

```bash
# 进入测试目录
cd 测试模型能力

# 如果需要下载模型
python 1_下载模型.py

# 测试纯文本对话
python 2_测试模型.py

# 测试图像分析
python 3_测试多模态模型.py
```

### 路径说明

所有脚本已经配置为使用相对路径，可以在 `测试模型能力` 文件夹内直接运行：
- 模型路径: `../models/Lingshu-7B`
- 图像路径: `../test_images/`

这样设计可以确保脚本在此文件夹内运行时，能正确访问项目根目录下的资源。

## 注意事项

1. 请确保已安装所需依赖包
2. 首次运行需要先下载模型
3. 推荐使用量化模式以减少显存占用
